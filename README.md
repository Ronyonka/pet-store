# Pet Store (Bark-n-Bites)

## Installations

- Next.js:
  ```
  npx create-next-app@latest .
  ```
- [Zustand:](https://zustand.docs.pmnd.rs/getting-started/introduction)
  ```
  npm install zustand
  ```
- [Zod:](https://zod.dev/)
  ```
  npm install zod
  ```
- [Prisma:](https://www.prisma.io/docs)

  ```
  npm install prisma -D
  npm install @prisma/client
  ```
